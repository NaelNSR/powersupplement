# 💪 Power Supplements – Gym Supplements Web App

## 📖 Description
A responsive web application for a gym supplements store built with React.  
Users can browse products, learn about the company, and contact support.

## 🚀 Live Demo
[View the live app here](https://powersupplement.vercel.app/)  

## 🛠️ Technologies Used
- React
- React Router
- CSS
- Git & GitHub
- Vercel

## 📦 Setup Instructions
1. Clone the repository:  
   `git clone https://github.com/NaelNSR/powersupplement.git`
2. Navigate into the folder:  
   `cd powersupplement`
3. Install dependencies:  
   `npm install`
4. Start the development server:  
   `npm start`
5. Open `http://localhost:3000` in your browser.
